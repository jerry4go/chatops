默认自带的一问一答回复脚本的路径在

scripts/example.coffee

默认是没有启用的，需要把注释去掉即可

可以直接输入 help
```
momo adapter - Reply with the adapter
momo animate me <query> - The same thing as `image me`, except adds a few parameters to try to return an animated GIF instead.
momo echo <text> - Reply back with <text>
momo help - Displays all of the help commands that this bot knows about.
momo help <query> - Displays all help commands that match <query>.
momo image me <query> - The Original. Queries Google Images for <query> and returns a random top result.
momo map me <query> - Returns a map view of the area returned by `query`.
momo mustache me <url|query> - Adds a mustache to the specified URL or query result.
momo ping - Reply with pong
momo pug bomb N - get N pugs
momo pug me - Receive a pug
momo the rules - Make sure hubot still knows the rules.
momo time - Reply with current time
momo translate me <phrase> - Searches for a translation for the <phrase> and then prints that bad boy out.
momo translate me from <source> into <target> <phrase> - Translates <phrase> from <source> into <target>. Both <source> and <target> are optional
ship it - Display a motivation squirrel
```
会自带一些回复响应
